Sistema Automatizado de Sorteos por WhatsApp

Una solución integral e inteligente que automatiza la recepción, validación y registro de tickets de sorteo a través de WhatsApp, brindando una mejor experiencia para el usuario y control total para la administración.

Flujo de Experiencia del Usuario (WhatsApp)

1

Registro Ágil sin Fricción

En el primer contacto, el bot solicita la foto de la cédula o el número de manera amigable para crear el perfil, obteniendo el teléfono en segundo plano automáticamente. Una vez registrado, el sistema recordará al usuario para futuras interacciones.

2

Identificación y Extracción

Al recibir la imagen, el sistema identifica si el ticket es Betplay o Chance y extrae automáticamente los datos clave (ID, valor, identificación) para su procesamiento.

3

Identificación Automatizada

El sistema cruza la información extraída con la base de datos para validar si el ticket ya está registrado y confirma si cumple con las condiciones para participar.

4

Control Detallado de Registros

Visualiza el detalle completo de cada participación con el ID de transacción, identificación del apostador y el valor total del ticket, con la posibilidad de implementar filtrado por tipo de ticket o por valor.

5

Dashboard y Nube Administrativa

Toda la información extraída y validada es guardada de forma segura y clasificada automáticamente por sorteo en un panel de administración personalizado. Además, cuentas con 15 GB de almacenamiento inicial gratuito disponible para todos tus sorteos.
